# ⚠️ 切换到真实交易模式 - 重要警告

## ⚠️ 重要警告

**切换到真实交易模式后，程序将执行真实交易，会消耗您的真实资金！**

### 风险提示

1. **资金损失风险**：跟单交易可能导致资金损失
2. **不可逆操作**：交易一旦执行，无法撤销
3. **市场风险**：市场价格波动可能导致亏损
4. **技术风险**：网络延迟、API 错误等可能导致意外损失

### 切换到真实交易模式前的检查清单

- [ ] 已充分测试（至少运行 24 小时以上）
- [ ] 确认网络连接稳定
- [ ] 确认钱包已在 Polymarket 注册
- [ ] 确认账户有足够的 USDC 余额
- [ ] 确认跟单金额设置合理（从小额开始）
- [ ] 已理解并接受所有风险
- [ ] 已准备好承担可能的损失

### 建议的配置

首次使用真实交易，建议：

```javascript
followSettings: {
  autoFollow: true,
  dryRun: false,  // ⚠️ 真实交易模式
  sizeScale: 0.05,  // 跟单5%（从小额开始）
  maxSizePerTrade: 10,  // 最大单笔$10（小额测试）
  minTradeSize: 1,  // 最小$1
}
```

## 切换步骤

1. **备份当前配置**
   ```bash
   cp config.js config.js.backup
   ```

2. **修改 config.js**
   ```javascript
   dryRun: false,  // 改为 false
   ```

3. **重新运行程序**

## 紧急停止

如果出现问题，立即：
- 按 `Ctrl+C` 停止程序
- 或使用 `pm2 stop polybot`（如果使用 PM2）

## 安全建议

1. ✅ 从小金额开始测试
2. ✅ 定期检查账户余额和持仓
3. ✅ 设置合理的止损
4. ✅ 不要使用超过您能承受损失的资金
5. ✅ 定期备份配置文件
