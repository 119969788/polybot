## 三步运行

### 1. 配置项目（如果需要）

默认配置已设置为**只读模式**（不执行交易），可以直接运行查看钱包信息。

如果需要跟单功能，编辑 `config.js`：

```javascript
// config.js
sdk: {
  privateKey: process.env.POLYMARKET_PRIVATE_KEY || '', // 私钥（可选）
}
followSettings: {
  autoFollow: false,  // 当前为 false，只查看信息
  dryRun: true,  // 测试模式，不会真实交易
}
```

### 2. 运行项目

#### 方式一：直接运行（推荐测试）

```bash
npm start
```

#### 方式二：开发模式（代码更改自动重启）

```bash
npm run dev
```

## 运行模式

### 📖 只读模式（当前默认）

查看钱包信息，不执行交易：

```bash
npm start
```

配置：`autoFollow: false`（已设置）

### 🧪 测试模式

模拟交易，不消耗真实资金：

```javascript
// config.js
followSettings: {
  autoFollow: true,
  dryRun: true,  // ✅ 测试模式
  sizeScale: 0.1,  // 跟单10%
}
```

### 💰 真实交易模式

⚠️ **仅在充分测试后使用！**

```javascript
// config.js
followSettings: {
  autoFollow: true,
  dryRun: false,  // ⚠️ 真实交易
}
```

## 遇到问题？

### 问题：显示"未找到配置文件"

**解决**：确保 `config.js` 存在（已存在，可直接使用）

### 问题：显示"无法连接"

**解决**：检查网络连接，确保能访问 GitHub 和 Polymarket

### 问题：显示"SDK 初始化失败"

**解决**：
```bash
# 重新安装依赖
rm -rf node_modules package-lock.json
npm install
```

### 问题：想要后台运行

**解决**：使用 PM2
```bash
npm install -g pm2
pm2 start index.js --name polybot
pm2 logs polybot
```
